# README #

My XMonad Configuration. Minimalistic and functional.

### How do I get set up? ###

* Install XMonad.
* Clone this repo.
* Move it to /home/username/.xmonad
* Run XMonad 

### Features ###

* Extensive GridSelect usage
* PromptShell
* TabbedLayout
